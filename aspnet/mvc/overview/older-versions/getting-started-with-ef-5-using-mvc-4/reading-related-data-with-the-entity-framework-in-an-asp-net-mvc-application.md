## <a name="lazy-eager-and-explicit-loading-of-related-data"></a><span data-ttu-id="3ef91-116">Отложенная, упреждающая и явная загрузка связанных данных</span><span class="sxs-lookup"><span data-stu-id="3ef91-116">Lazy, Eager, and Explicit Loading of Related Data</span></span>

<span data-ttu-id="3ef91-117">Существует несколько способов, что платформа Entity Framework можно загрузить связанные данные в свойства навигации сущности:</span><span class="sxs-lookup"><span data-stu-id="3ef91-117">There are several ways that the Entity Framework can load related data into the navigation properties of an entity:</span></span>

- <span data-ttu-id="3ef91-123">*Безотложная загрузка*.</span><span class="sxs-lookup"><span data-stu-id="3ef91-123">*Eager loading*.</span></span> <span data-ttu-id="3ef91-124">При чтении сущности связанные данные извлекаются вместе с ней.</span><span class="sxs-lookup"><span data-stu-id="3ef91-124">When the entity is read, related data is retrieved along with it.</span></span> <span data-ttu-id="3ef91-125">Обычно такая загрузка представляет собой одиночный запрос с соединением, который получает все необходимые данные.</span><span class="sxs-lookup"><span data-stu-id="3ef91-125">This typically results in a single join query that retrieves all of the data that's needed.</span></span> <span data-ttu-id="3ef91-126">Безотложная загрузка задаются с помощью `Include` метод.</span><span class="sxs-lookup"><span data-stu-id="3ef91-126">You specify eager loading by using the `Include` method.</span></span>
