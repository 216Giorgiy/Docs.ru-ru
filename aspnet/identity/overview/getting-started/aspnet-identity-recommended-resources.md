- Вопрос. можно изменить утверждения в ASP.NET Identity с OWIN после вызова вход Ответ: SignIn вызова именно то, что должен делать, если требуется изменить утверждения для пользователя. По сути, это вызывает ClaimsIdentity, который будет сериализован в файл cookie, поэтому вы видите новых утверждений, которые отображаются при последующих запросах.
