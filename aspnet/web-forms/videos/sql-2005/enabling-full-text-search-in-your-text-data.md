<a name="enabling-full-text-search-in-your-text-data"></a>Включение полнотекстового поиска в данных текст
====================
по [Microsoft](https://github.com/microsoft)

Компонент Full-Text search позволяет сохранить большой части текста — даже документов Microsoft Word или другие форматы файлов - в базу данных и выполнения сложных запросов на основе этих текстовых данных во время ожидается высокую производительность. На этом занятии рассматривается, как настроить Full-Text search, уделяя особое внимание особенности настройки в SQL Server Express. Мы переместите создание полнотекстовых индексов, Настройка параметров заполнения индекса и выполнения запросов, которые демонстрируют возможности предиката CONTAINS и его множество вариантов.
