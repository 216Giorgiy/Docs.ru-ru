<a name="how-do-i-use-the-aspnet-ajax-maskededit-controls"></a>[Инструкции:] С помощью элементов управления ASP.NET AJAX MaskedEdit?
====================
по [(Joe Stagner)](https://github.com/JoeStagner)

Набор элементов управления ASP.NET AJAX предоставляет элемент управления MaskedEdit и управления MaskedEditValidator, который совместно запрашивать стиля и проверки данных, вводимых пользователем в текстовое поле. Это позволяет веб-странице предоставить пользователю возможность немедленного визуальную обратную связь на наличие данных, введенных допустимый тип и формат.
