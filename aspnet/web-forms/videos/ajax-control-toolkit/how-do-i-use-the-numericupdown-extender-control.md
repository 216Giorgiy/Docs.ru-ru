<a name="how-do-i-use-the-numericupdown-extender-control"></a>[Инструкции:] Используйте элемент управления NumericUpDown расширителя?
====================
по [(Joe Stagner)](https://github.com/JoeStagner)

В этом видеоролике рассматриваются расширитель NumericUpDown из элементов управления ASP.NET AJAX. Мы узнаем, как расширения NumericUpDown добавляет «вверх» и «вниз», в стандартный элемент управления ASP.NET TextBox. Кнопки позволяют увеличить или уменьшить число перебирать список известных значений и вызова веб-службы для получения следующего значения. В этом случае разработчик может полностью управлять значения, которые пользователь может отправить на веб-приложение.
