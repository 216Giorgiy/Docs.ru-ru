<a name="how-do-i-use-the-aspnet-ajax-resizablecontrol-extender"></a>[Инструкции:] Использовать расширитель ASP.NET AJAX ResizableControl?
====================
по [(Joe Stagner)](https://github.com/JoeStagner)

Расширитель ResizableControl из элементов управления ASP.NET AJAX позволяет разработчику указать регион веб-страницы, пользователь может динамически изменить размер.
