<a name="how-do-i-use-the-aspnet-ajax-updateprogress-control"></a>[Инструкции:] Использование элемента управления UpdateProgress ASP.NET AJAX?
====================
по [(Joe Stagner)](https://github.com/JoeStagner)

В приложении AJAX обычно имеется заметное пауза между временем браузер отправляет запрос сети и времени, браузер получает ответ. Для оповещения пользователя о том, что сетевую беседу, предпринимаются, ASP.NET AJAX UpdateProgress элемент управления предоставляет оперативные сведения по отображению анимированное изображение, текст «загрузка» или других HTML, исчезнет после получения ответа сети.
