<a name="how-do-i-use-the-ftpwebrequest-class-to-create-an-ftp-connection-to-a-ftp-server"></a>[Инструкции:] Класс FtpWebRequest используется для создания подключения к FTP для FTP-сервера
====================
по [Крис пиксел](https://twitter.com/chrispels)

В этот видеоролик пиксел Крис показывает, как использовать класс FtpWebRequest для создания FTP-соединение и передачи файла. Во-первых создайте простое .aspx для выбора файла для передачи на FTP-сервер. Далее в разделе Основные методы и свойства класса FtpWebRequest, который должен быть настроен для установления подключения к FTP. Затем посмотрите, как получить содержимое локального файла, файл может передаваться с помощью StreamReader и StreamWriter записываемый FtpWebRequest. Наконец для определения результата передачу получается FtpWebResponse. Этот подход можно использовать программным путем передачи файлов для веб-приложения.
