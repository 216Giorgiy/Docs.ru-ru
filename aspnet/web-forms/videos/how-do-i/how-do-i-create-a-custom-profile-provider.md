====================
по [Крис пиксел](https://twitter.com/chrispels)

В этом видеоролике рассказано, как создать настраиваемый профиль поставщика для хранения и извлечения данных, связанных с профилем пользователя в таблицах SQL Server отдельно от стандартного поставщика членства ASP.NET aspnet\_таблице профилей. Принципы, описанные в этом видео будет применяются при создании профиля поставщиков для других баз данных, таких как Access и Oracle. После изучения способов создания поставщика пользовательских профилей рассказано, как установить поставщик веб-сайт и просмотреть демонстрацию загрузки и сохранения данных профиля.
