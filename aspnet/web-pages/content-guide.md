<a name="webmatrix-content-guide"></a>Руководство по содержимому WebMatrix
====================

> [!NOTE] 
> WebMatrix не рекомендуется использовать в интегрированной среде разработки для веб-страниц ASP.NET. Используйте [Visual Studio](xref:aspnet/web-pages/overview/getting-started/program-asp-net-web-pages-in-visual-studio) или [кода Visual Studio](https://code.visualstudio.com/).

WebMatrix — это набор средств, который предоставляет простой способ создания веб-сайтов для Microsoft Windows. Он включает в себя сервер IIS Express (веб-сервер разработки), ASP.NET (веб-платформа программирования) и SQL Server Compact (встроенную базу данных). Краткий обзор WebMatrix см. в разделе [Обзор WebMatrix](https://www.microsoft.com/web/webmatrix/). Исключительные возможности WebMatrix см. в разделе [функции WebMatrix](https://www.microsoft.com/web/webmatrix/features/).
