# <a name="aspnet-core-middleware-extensibility-sample"></a>Пример расширяемости ПО промежуточного слоя ASP.NET Core

Этот пример описывает использование интерфейсов [IMiddleware](https://docs.microsoft.com/dotnet/api/microsoft.aspnetcore.http.imiddleware) и [IMiddlewareFactory](https://docs.microsoft.com/dotnet/api/microsoft.aspnetcore.http.imiddlewarefactory). Пример демонстрирует использование функции, описанных в статье [Фабричный метод активации ПО промежуточного слоя в ASP.NET Core](https://docs.microsoft.com/aspnet/core/fundamentals/middleware/middleware-extensibility).
