В этом руководстве описывается модель MVC ASP.NET Core с контроллерами и представлениями. Страницы Razor — это новая альтернатива в ASP.NET Core 2.0, представляющая собой основанную на страницах модель программирования, которая упрощает и повышает эффективность создания пользовательского веб-интерфейса. Версия этого руководства для Razor представлена в разделе [Руководство по страницам Razor](xref:mvc/razor-pages/index).
