## <a name="partial-view-discovery"></a><span data-ttu-id="d40c8-159">Обнаружение частичного представления</span><span class="sxs-lookup"><span data-stu-id="d40c8-159">Partial view discovery</span></span>

```cshtml
<partial name="_ViewName.cshtml" />

// Locate the view using a relative path
<partial name="../Account/_LoginPartial.cshtml" />
```

<span data-ttu-id="d40c8-162">В предыдущем примере используется вспомогательная функция тега частичного представления, для которой требуется ASP.NET Core 2.1 или более поздней версии.</span><span class="sxs-lookup"><span data-stu-id="d40c8-162">The preceding example uses the Partial Tag Helper, which requires ASP.NET Core 2.1 or later.</span></span> <span data-ttu-id="d40c8-163">В следующем примере используются асинхронные вспомогательные методы HTML для выполнения той же задачи.</span><span class="sxs-lookup"><span data-stu-id="d40c8-163">The following example uses asynchronous HTML Helpers to accomplish the same task.</span></span>
